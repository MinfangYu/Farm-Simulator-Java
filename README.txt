
README
In this game, you are a farmer lost in an alien planet. You have to earn enough money to buy a spaceship in a limited time to return home.
To build source code instruction:
Open Eclipse >>File>>Import>>General>> Projects from folder or Archive>>Next>>Directory>>Choose  Avengers: Homesickness>> OK>>Finish

To run program in eclipse
Double click story in eclipses and run

To run Program in terminal
java -jar wli91_myu40_FarmingSimulator.jar
 

Java version is 1.8.
